# VIM note

## 開啟
`vim -p file` 以分頁方式開啟
`vim -[oO] file` 以分割方式開啟`




# 移動
`zz` 將目前的所在的游標至中。

## 搜尋
ignorecase 無視case
smartcase 在你搜尋有大寫時，則會無效ignorecase
`:noh` 關掉搜尋顯示的highlight





## 在vim 中使用指定

`:! <command>`
`:r ! <command>` 在目前位置貼上指定內容


## 折疊行
使用`zf`折疊選取行，用`l`展開。
可在後面加ia


## tab
`CTRL-w [sv]` 分割視窗
`CTRL-w [<>+-]` 調整當前分割視窗的長寬
